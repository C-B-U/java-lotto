# 기능 목록

### Member

- [x] 로또 구입 금액 입력
- [x] 당첨 번호 입력
- [x] 보너스 번호 입력

### Lotto

- [x] 랜덤된 로또 번호 생성 후 출력
- [x] 입력 숫자와 로또 번호 비교
- [x] 비교된 결과 표 출력
- [x] 수익률 출력


# 예외 처리
사용자가 잘못된 값을 입력할 경우 IllegalArgumentException 발생, "[ERROR]"로 시작하는 에러 메시지를 출력 후 종료
- [x] 로또 구입 금액은 1000 단위로 나누어 떨어지는 숫자
- 당첨 번호는 중복 안되는 1~45까지의 6개의 숫자
- 보너스 점수는 1~45까지의 숫자 한개
